HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar páginas da web. Ele permite definir títulos, parágrafos, imagens, links e muito mais. Com HTML, é possível criar uma base sólida para um site, organizando os elementos de forma hierárquica e semântica. Junto com CSS e JavaScript, HTML é um dos pilares do desenvolvimento web moderno.

CSS (Cascading Style Sheets) é a linguagem utilizada para estilizar páginas HTML. Ele permite definir cores, fontes, espaçamentos, alinhamentos e até animações. Com CSS, é possível tornar um site visualmente atraente e responsivo, garantindo que ele se adapte a diferentes dispositivos. O CSS pode ser aplicado de forma inline, interna ou externa, sendo a última a mais recomendada para manter um código organizado.

JavaScript é a linguagem de programação que torna as páginas da web interativas. Com ele, é possível criar desde pequenas animações até aplicações completas. JavaScript é amplamente usado para manipular elementos HTML, responder a eventos do usuário e se comunicar com servidores. Graças às suas bibliotecas e frameworks, como React e Vue, JavaScript é essencial para o desenvolvimento web moderno.
